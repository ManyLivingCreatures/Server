# ManyLivingCreatures MMO Server | 从生之门 MMO 服务器

**衍生自《罗小黑战记》TV 版中虚构游戏《众生之门》的非商业二创游戏**  
*(详细信息请参考 "引用说明 & 致谢" 章节)*

> 感谢木头及其团队用最温柔的笔触, 最温暖的故事, 创造了《罗小黑》这个关于罗小黑, 人, 妖的故事

**本项目为 MMO 服务器, 客户端侧 (即本体) 请前往 [这里](https://github.com/ManyLivingCreatures/GameGodot)**

## 技术栈

本项目使用 [.NET Core](https://dotnet.microsoft.com/) + [ASP.NET Core](https://github.com/dotnet/aspnetcore) + [Orleans](https://github.com/dotnet/orleans)
